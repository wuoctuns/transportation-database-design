# 🚚 Transportation Database Design

## 📖 Overview

This project was developed as part of the **Database** course at the **University of Economics Ho Chi Minh City (UEH)**.

The project focuses on designing and implementing a relational database system for a transportation and logistics company. It supports the management of customers, products, orders, employees, deliveries, suppliers, and payment information while ensuring data integrity through constraints and business rules.

---

## 🎯 Objectives

- Design a normalized relational database.
- Apply database design principles and ERD modeling.
- Implement business rules using SQL Server.
- Improve database consistency and integrity.
- Practice SQL programming and database management.

---

## 🛠 Technologies

- Microsoft SQL Server
- SQL (DDL, DML, DCL)
- Entity Relationship Diagram (ERD)
- Triggers
- Views
- Constraints
- Primary Keys & Foreign Keys

## 🚀 How to Use

1. Execute **create_tables.sql** to create the database schema.
2. Run **insert_data.sql** to populate sample data.
3. Execute **triggers.sql** to apply business rules.
4. Explore the database using Microsoft SQL Server Management Studio (SSMS).
